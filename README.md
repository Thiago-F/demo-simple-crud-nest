## Description

Simple crud using nest framework!

## Tools

- Bcrypt
- Jwt
- Prisma

## Installation

```bash
$ npm install
$ npm run prisma:build
$ npm run build
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test:watch
```
